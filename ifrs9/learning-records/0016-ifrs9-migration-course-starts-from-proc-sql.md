# IFRS9 migration course starts from PROC SQL

The user wants a multi-session course on converting IFRS9 SAS 9 code to Snowflake, beginning with familiar `PROC SQL` and gradually moving toward Snowflake SQL, Snowpark, orchestration, and audit-ready migration controls. Future lessons should stay hands-on, example-led, and tied to IFRS9-style staging, stage allocation, ECL aggregation, reconciliation, and governance evidence.
