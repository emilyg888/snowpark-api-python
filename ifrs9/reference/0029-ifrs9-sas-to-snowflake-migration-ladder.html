10-Lesson Sequence

  1. Translate a simple IFRS9 PROC SQL staging query.
  2. Port SAS numeric date filters and reporting cutoffs from AS_OF_DT.
  3. Rebuild joins for loan account, customer, collateral, and parameter tables.
  4. Convert risk segmentation and stage allocation logic.
  5. Aggregate PD, LGD, EAD, and ECL by segment.
  6. Replace SAS formats and macro variables with Snowflake reference tables.
  7. Move reusable transformations into Snowpark DataFrames.
  8. Add reconciliation checks against SAS outputs.
  9. Package the pipeline with stored procedures and task graphs.
  10. Prepare audit-ready migration evidence and extension points.

Rule Of Thumb

Port the calculation in SQL first when the SAS code is already relational. Move to Snowpark when the same pattern repeats, needs Python testability, or must become a reusable procedure.
