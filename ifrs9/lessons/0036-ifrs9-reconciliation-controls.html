IFRS9 Reconciliation Controls

Goal: build row-count and amount reconciliation checks between SAS and Snowflake outputs.

Migration quality is proven by differences, thresholds, and explanations.

Core Example

select
  s.CONTROL_NAME,
  s.SAS_VALUE,
  f.SNOWFLAKE_VALUE,
  f.SNOWFLAKE_VALUE - s.SAS_VALUE as DIFFERENCE
from SAS_CONTROLS s
join SNOWFLAKE_CONTROLS f using (CONTROL_NAME);

Hands-On Exercise

  1. Create controls for row count, total EAD, and total ECL.
  2. Join SAS and Snowflake control tables.
  3. Flag differences outside tolerance.
