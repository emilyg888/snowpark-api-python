IFRS9 Formats And Macros

Goal: replace SAS formats and macro variables with Snowflake reference tables and parameters.

A format or macro often hides business logic. In this dataset, start with %let as_of_dt, %let run_id, and the PD/LGD parameter CSV.

Core Example

create or replace table IFRS9_STAGE_RULES (
  RULE_NAME string,
  MIN_DPD number,
  STAGE number
);

Hands-On Exercise

  1. List every SAS macro variable used in the query.
  2. Classify each as environment, cutoff, threshold, or business rule.
  3. Move business thresholds into a reference table.
