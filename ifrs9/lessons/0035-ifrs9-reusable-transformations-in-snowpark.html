IFRS9 Reusable Transformations In Snowpark

Goal: move repeated SQL patterns into small Snowpark DataFrame functions.

Use Snowpark when reuse, tests, or procedural composition are more important than one-off SQL readability.

Core Example

def add_stage(df):
    return df.with_column(
        "IFRS9_STAGE",
        when(col("DAYS_PAST_DUE") >= 90, lit(3))
        .when(col("DAYS_PAST_DUE") >= 30, lit(2))
        .otherwise(lit(1)),
    )

Hands-On Exercise

  1. Extract one repeated case when block.
  2. Write it as a Snowpark function.
  3. Save the output and compare it to the SQL version.
