Saving Snowpark DataFrames

Quick reference for materializing DataFrames as Snowflake tables.

Core Pattern

df.write.save_as_table("DATABASE.SCHEMA.TABLE_NAME", mode="overwrite")

Mode Cheatsheet

overwrite
Replace the target table.
append
Add rows to the target table.
errorifexists
Fail if the target table exists.
ignore
Skip the write if the target table exists.

Boundary Rule

Keep writers out of reusable builder functions. A write is a Snowflake state change, so name it and test it separately.
