Flattening Arrays in Snowpark

Quick reference for turning nested array elements into rows.

Use When

  • An array can contain multiple meaningful values.
  • Each value should become filterable, countable, or displayable as a row.
  • The output row grain should change from one source object to one nested element.

Core Pattern

from snowflake.snowpark.functions import col, flatten
from snowflake.snowpark.types import StringType

flattened = (
    car_sales
    .select(
        col("SRC")["dealership"].cast(StringType()).as_("DEALERSHIP"),
        flatten(col("SRC")["vehicle"][0]["extras"])
    )
    .select(
        "DEALERSHIP",
        col("VALUE").cast(StringType()).as_("EXTRA")
    )
)

Remember

flatten exposes metadata fields. The nested element itself is in VALUE.
