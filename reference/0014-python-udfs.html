Python UDFs

Quick reference for Python functions called from SQL expressions.

Use For

  • Scalar transformations.
  • Reusable row-level business rules.
  • Logic that returns one value for each input row.

Core Shape

create or replace function NAME(input string)
returns string
language python
runtime_version = '3.12'
handler = 'handler_name'
as
$$
def handler_name(input):
    return input
$$;

Boundary Rule

Use a UDF for expression logic. Use a stored procedure when the job needs to write tables or orchestrate multiple steps.
