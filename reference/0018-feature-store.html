Feature Store

Quick reference for Snowflake Feature Store objects in the lab.

Objects

FeatureStore
Client tied to database, schema, and warehouse.
Entity
Business subject and join keys, such as LOAN_ID.
FeatureView
Registered feature DataFrame with timestamp and refresh cadence.

Core Pattern

entity = Entity(name="LOAN_ENTITY", join_keys=["LOAN_ID"])
fs.register_entity(entity)
fv = FeatureView(name="Mortgage_Feature_View", entities=[entity], feature_df=feature_df, timestamp_col="TIMESTAMP")
fs.register_feature_view(fv, version=VERSION_NUM, overwrite=True)

Rule

Define entity grain before registering a feature view. The feature view should compute features at that grain.
