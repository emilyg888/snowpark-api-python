Registry and Monitoring

Quick reference for model versioning, inference, and observability.

Lifecycle

registry = Registry(session=session, database_name=DB, schema_name=SCHEMA)
mv = registry.log_model(..., options={"enable_explainability": True})
predictions = mv.run(test, function_name="predict")
registry.get_model(model_name).default = "XGB_Optimized"

Rule

A trained model is not production-ready until it is versioned, callable, promoted deliberately, and monitored against baseline behavior.
