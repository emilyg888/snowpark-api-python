Snowpark Result Boundaries

A quick reference for deciding when a lazy DataFrame becomes local output.

Rule

Transform broadly in Snowflake. Return only small, intentional results to Python, pandas, notebooks, or Streamlit.

Common Actions

show()
Execute and print a preview for humans.
collect()
Execute and return Snowpark Row objects.
to_pandas()
Execute and return a local pandas DataFrame.
to_pandas_batches()
Execute and stream pandas batches.

Safe Pattern

summary = (
    df
    .filter(...)
    .group_by(...)
    .agg(...)
)

summary.limit(20).show()
chart_df = summary.to_pandas()
