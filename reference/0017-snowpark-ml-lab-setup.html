Snowpark ML Lab Setup

Quick reference for binding a notebook to the lab context and loading raw data.

Context

session = get_active_session()
session.use_role("E2E_SNOW_MLOPS_ROLE")
session.use_database("E2E_SNOW_MLOPS_DB")
session.use_schema("MLOPS_SCHEMA")
session.use_warehouse("E2E_SNOW_MLOPS_WH")

Load Pattern

df_pandas = pd.read_csv("MORTGAGE_LENDING_DEMO_DATA.csv.zip")
session.write_pandas(df_pandas, "MORTGAGE_LENDING_DEMO_DATA", auto_create_table=True, overwrite=True)
df = session.table("MORTGAGE_LENDING_DEMO_DATA")

Rule

Use Snowpark DataFrames after the raw data has landed. Treat pandas as an ingestion bridge here, not the main processing engine.
