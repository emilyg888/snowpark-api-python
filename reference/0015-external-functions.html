External Functions

Quick reference for SQL functions backed by remote HTTPS services.

Call Path

SQL query
  -> external function
  -> API integration
  -> cloud proxy
  -> remote service
  -> result returned to Snowflake

Use Only When

  • The logic must call a system outside Snowflake.
  • A normal SQL expression, Snowpark expression, or Python UDF is not enough.
  • The team can own network security, service reliability, cost, and latency.

Boundary Rule

An external function adds a network boundary. Design and review the contract before creating infrastructure.
