Snowpark Derived Columns

Quick reference for adding columns with DataFrame expressions.

Imports

from snowflake.snowpark.functions import col, lit, when

Add One Column

df2 = df.with_column("IS_HIGH_VALUE", col("O_TOTALPRICE") >= lit(30000))

Add Conditional Column

df2 = df.with_column(
    "ORDER_URGENCY",
    when(col("O_ORDERPRIORITY") == lit("1-URGENT"), lit("URGENT"))
    .otherwise(lit("STANDARD"))
)

Replace Existing Column

df2 = df.with_column("O_TOTALPRICE", col("O_TOTALPRICE") / lit(1000))

Select With Alias

df2 = df.select(
    col("O_ORDERKEY").as_("ORDER_ID"),
    col("O_TOTALPRICE").as_("TOTAL_PRICE")
)

Rules

  • Use col("NAME") to reference a column.
  • Use lit(value) for constants inside expressions.
  • with_column returns a new DataFrame; it does not mutate the original.
  • Reusing an existing column name replaces that column in the returned DataFrame.
