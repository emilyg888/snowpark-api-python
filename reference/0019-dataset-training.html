Dataset and Training

Quick reference for generating a training dataset and fitting the baseline model.

Pipeline

ds = fs.generate_dataset(...)
ds_sp = ds.read.to_snowpark_dataframe()
ds_sp_ohe = OneHotEncoder(...).fit(ds_sp).transform(ds_sp)
train, test = ds_sp_ohe.random_split(weights=[0.70, 0.30], seed=0)
train_pd = train.fillna(0).to_pandas()

Rule

Keep feature retrieval and preprocessing in Snowflake. Convert to pandas only when the training library requires local data.
