Snowpark Grouping and Aggregation

Quick reference for turning row-level Snowpark data into grouped summaries.

Imports

from snowflake.snowpark.functions import col
from snowflake.snowpark.functions import count, sum as sum_, avg, min as min_, max as max_

Basic Pattern

summary = (
    df
    .group_by("O_ORDERPRIORITY")
    .agg(
        count("*").as_("ORDER_COUNT"),
        sum_("O_TOTALPRICE").as_("TOTAL_REVENUE"),
        avg("O_TOTALPRICE").as_("AVG_ORDER_VALUE")
    )
    .sort(col("TOTAL_REVENUE").desc())
)

Multiple Group Keys

summary = (
    df
    .group_by("O_ORDERSTATUS", "O_ORDERPRIORITY")
    .agg(count("*").as_("ORDER_COUNT"))
)

Rules

  • group_by(...) creates grouped rows, similar to SQL GROUP BY.
  • agg(...) chooses the summary measures for each group.
  • Alias aggregate columns with as_(...) so downstream code has stable names.
  • Sorting after aggregation makes summaries readable.
  • The query is still lazy until an action such as show() or collect().
