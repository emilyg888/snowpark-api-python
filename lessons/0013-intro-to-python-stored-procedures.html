Intro to Python Stored Procedures

Goal: recognize when Snowpark code should run as a callable procedure inside Snowflake.

A Python stored procedure is a named Snowflake object whose handler runs inside Snowflake. Use it when the work is a pipeline step, a refresh command, or a controlled table-changing operation.

The Mental Model

Local script
Python runs on your machine and sends DataFrame plans to Snowflake.
Stored procedure
Python handler code runs inside Snowflake with a Snowpark session passed in.
CALL
SQL invokes the procedure and receives a status value or result.

Read This Code

create or replace procedure REFRESH_VEHICLE_PRICES()
returns string
language python
runtime_version = '3.12'
packages = ('snowflake-snowpark-python')
handler = 'refresh_vehicle_prices'
as
$$
from snowflake.snowpark.functions import col
from snowflake.snowpark.types import IntegerType, StringType

def refresh_vehicle_prices(session):
    car_sales = session.table("SNOWPARK_SAMPLE_DATA.PUBLIC.CAR_SALES")

    vehicle_prices = (
        car_sales
        .select(
            col("SRC")["dealership"].cast(StringType()).as_("DEALERSHIP"),
            col("SRC")["vehicle"][0]["make"].cast(StringType()).as_("MAKE"),
            col("SRC")["vehicle"][0]["model"].cast(StringType()).as_("MODEL"),
            col("SRC")["vehicle"][0]["year"].cast(IntegerType()).as_("YEAR"),
            col("SRC")["vehicle"][0]["price"].cast(IntegerType()).as_("PRICE")
        )
    )

    vehicle_prices.write.save_as_table(
        "SNOWPARK_SAMPLE_DATA.PUBLIC.VEHICLE_PRICES",
        mode="overwrite"
    )
    return "VEHICLE_PRICES refreshed"
$$;

call REFRESH_VEHICLE_PRICES();

The handler receives session from Snowflake. The write is intentional: this procedure refreshes a table and returns a short status string.

Use It For

  • Refreshing a table from repeatable Snowpark logic.
  • Running a controlled multi-step pipeline inside Snowflake.
  • Creating work that can later be called by SQL, tasks, or operational runbooks.
